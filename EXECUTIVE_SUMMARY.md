## 🎨 Design System

### Color Palette
```
Primary:    #2563eb (Blue)
Gradient:   Blue → Indigo
Accent:     #06b6d4 (Cyan)
Success:    #10b981 (Green)
Neutral:    Slate 50-900
```

### Typography Scale
```
H1: 48px (sm) → 84px (lg)
H2: 36px (sm) → 60px (lg)
H3: 24px (sm) → 32px (lg)
Body: 16px (sm) → 20px (lg)
```

### Spacing Scale
```
xs: 4px    | sm: 8px    | md: 16px   | lg: 24px
xl: 32px   | 2xl: 48px  | 3xl: 64px  | 4xl: 96px
```
