## 💡 Customization Guide

### Colors
Edit `tailwind.config.js`:
```js
theme: {
  extend: {
    colors: {
      blue: { 600: '#2563eb' }
    }
  }
}
```

### Animations
Edit `src/lib/animations.tsx`:
```tsx
export const fadeInUp = {
  initial: { opacity: 0, y: 20 },
  animate: { opacity: 1, y: 0 },
  transition: { duration: 0.6 }
};
```
