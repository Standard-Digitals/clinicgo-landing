## 🚀 Getting Started

### Installation
```bash
npm install
npm run dev
```

### Customization
- Edit colors in `tailwind.config.js`
- Modify animations in `src/lib/animations.tsx`
- Update content in page components
- Add new sections as needed

### Deployment
```bash
npm run build
# Deploy to Vercel, Netlify, or AWS
```
