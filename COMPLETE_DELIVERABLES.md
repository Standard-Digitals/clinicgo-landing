## 🎓 Getting Started

### 1. Install Dependencies
```bash
npm install
```

### 2. Start Development Server
```bash
npm run dev
```

### 3. Open in Browser
```
http://localhost:5173
```

### 4. Build for Production
```bash
npm run build
```

### 5. Deploy
```bash
# Vercel
vercel

# Netlify
netlify deploy --prod --dir=dist

# AWS Amplify
amplify publish
```
